# 🐾 Adopet

Projeto desenvolvido com o objetivo de estudar e aplicar o tratamento de exceções em Java utilizando boas práticas de desenvolvimento.

## 📖 Sobre o projeto

O **Adopet** foi desenvolvido como exercício prático durante um curso de Java, com foco no entendimento do funcionamento das exceções e na forma correta de utilizá-las em aplicações.

Durante o desenvolvimento foram explorados conceitos como:

* Utilização das exceções nativas do Java.
* Criação de exceções personalizadas para regras de negócio.
* Diferença entre **Checked Exceptions** (`Exception`) e **Unchecked Exceptions** (`RuntimeException`).
* Centralização do tratamento de erros através de um **Global Exception Handler**.
* Boas práticas para retorno de mensagens de erro em aplicações.

O objetivo não era construir uma aplicação completa, mas compreender como estruturar um projeto de forma mais limpa e organizada quando ocorrem falhas durante a execução.

## 🚀 Conceitos praticados

* Tratamento de exceções (`try`, `catch`, `throw` e `throws`)
* Exceções verificadas (Checked Exceptions)
* Exceções não verificadas (Unchecked Exceptions)
* Exceções personalizadas
* `RuntimeException`
* `Exception`
* Tratamento global de exceções
* Organização em camadas (Controller, Service e Exception Handler)

## 🛠️ Tecnologias utilizadas

* Java
* Spring Boot
* Maven

## 🎯 Objetivo de aprendizado

Este projeto teve como finalidade consolidar os conhecimentos sobre tratamento de exceções em aplicações Java, entendendo quando utilizar as exceções fornecidas pela linguagem, quando criar exceções específicas para regras de negócio e como centralizar o tratamento de erros para tornar a aplicação mais organizada, legível e de fácil manutenção.
